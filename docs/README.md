# ✅ 게임 방식
1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임
- 같은 수가 같은 자리에 있으면 스트라이크
- 다른 자리에 있으면 볼
- 같은 수가 전혀 없으면 낫싱
- 컴퓨터는 1에서 9까지 서로 다른 임의의 수 3개 선택
- 컴퓨터가 선택한 3개의 숫자를 모두 맞히면 게임 종료
- 게임 종류 후 게임을 다시 시작하거나 완전히 종료할 수 있음
